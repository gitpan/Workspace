NAME
      upgrade -- Upgrade a workspace.

SYNOPSIS
      upgrade <workspace-file-names>

DESCRIPTION
    Makes a new version of <workspace> that is compatible with the installed
    version of Tk::Workspace.pm. The pre-upgrade workspace is saved with the
    extension ".old," and the text itself is saved to
    "<workspace-name>.txt."
